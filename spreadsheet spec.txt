

=== Spreadsheet Specification

Make excel spreadsheet with persistent column header
Column header text can wrap to allow narrower columns

Use different light colors to color just the cell (not the entire row) when it is NYS workday, non-NYS workday, vacation day, holiday day, work from home day, sick day

During RSU vesting period, color cell dark green

Display the following in day_info column
	On holiday day, display name of holiday
	On travel day, display departure location departure travel start time -> destination location destination travel end time
	All other days display actual location for that day
